## Finalizing

Start "Steam Big Picture Mode" or "Gaming Mode" (On SteamOS)

Open the Quick Access Menu (QAM) using the "..." button (on Steam Deck) or "Ctrl + 2" keys

Navigate to Decky (Plug-shaped icon) > "Decky Store" (top-right Shop-icon)

Search for and install "CSS Loader" and "Audio Loader"

Return to Decky > CSS Loader > Enable "MHOnDeck CSS Theme"

Return to Decky > Audio Loader > Enable "MHOnDeck SFX Pack"
